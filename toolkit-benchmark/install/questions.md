## Glossary
* Language Model
  usually a number of n-gram counts and probabilities e.g. in ARPA format
* Smoothing, Pruning, Discounting, Backoff
* Estimator
* Vocabulary
* Corpus
  usually provided as txt file with one sentence per line
* Sentence
* Sequence
* Word
* Token
